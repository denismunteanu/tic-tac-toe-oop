A two-player Tic Tac Toe game written in Python using object-oriented programming principles.

## Features
- Two-player local gameplay
- Input validation
- Win/tie detection
- Scoreboard across rounds
- Alternating starting player each round

## How to run
1. Make sure you have Python 3 installed.
2. Run:

python tic-tac-toe.py

## What I practiced
- Object-oriented programming (classes and objects)
- Encapsulation (Board, Player, ScoreBoard)
- Game loop and state management
